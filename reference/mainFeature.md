# Main Features

## Vision

OpenBayan provides unbiased semantic search across multiple sources of Islamic knowledge. We give users access to the sources, connections, and summaries so they can explore, verify, and summarize the material themselves.

## Core Features

### Search Your Way

Search using Arabic, English, Indonesian, Russian, translations, or transliteration.

Official translations and transliterations are limited. When a translation or transliteration is unavailable, AI-generated output may be provided with a clear warning.

### Multi-Source Knowledge

Search across the Qur'an, hadith, tafsir, aqidah, fiqh, and many books from classical and modern ulama.

Availability depends on whether the text is open, accessible, and ready to process.

### Rich Search Connections

Search is multi-layered and can use:

- Character, word, and sentence similarity.
- Semantic similarity through embeddings.
- Entity detection with nearby and strongly connected graph relations.
- Arabic root words, popular variations, and related graph connections.

### Custom Processing Pipelines

Each source, and even each specific book, can have a custom pipeline to improve graph data quality and source-specific search behavior.

### Rich Entity References

To improve categorization, entity detection, and relations, every entity can be connected with a Wikipedia page and web search results as extra references.

### Save And Note It Your Way

Save anything as an alamat, including search queries, sentences, ayat, hadith, categories, and entities.

Alamat can be collected into a majmu'. When users find insight after tadabbur, they can add notes directly to their alamat.

### File-Based Documents

Write a sahifah with BlockNote. Embed queries, ayat, hadith, categories, sentences, and even a majmu' of alamat. Add narration around those materials so the sahifah becomes a complete article.

If readers do not agree with the conclusion, they can explore the theme themselves through the embedded sources and connections.

## Future Features

### Presentation Mode

Use Reveal.js and Chalkboard directly from a sahifah. Choose which text to present and add extra narration as notes. Present online or download the result as a PDF.

### Offline Work

Write sahifah, create presentations, and manage alamat locally. Users can also download sources for offline local search.
